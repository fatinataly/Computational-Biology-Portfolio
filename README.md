## 👋 Hello, World (and all 3 billion base pairs of it)

I'm a **Computational Biologist** working at the intersection of multi-omics,
cancer genomics, and machine learning  with the stubborn belief that
the right pipeline, the right data, and enough compute time
can actually help beat cancer.

This portfolio is my attempt to organize years of work, scripts,
pipelines, and late-night analysis sessions into something that doesn't
look like `/home/user/final_v3_FINAL_real_this_time_2/`.
